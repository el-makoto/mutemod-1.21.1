# Discord Mute Mod by el-makoto

## English

a simple fabric mod for 1.21.1 for mute people on discord
 (the config save on config/discordmute.properties)
## Commands

set Token    /setmutetoken token

Mute channel    /muteall channel id

Mute discord user    /muteall player discorduser

Unmute channel    /unmuteall channel id

Unmute discord user    /unmuteall player discorduser
