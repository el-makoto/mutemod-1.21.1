# Discord Mute Mod by el-makoto

## English

a simple fabric mod for 1.21.1 for mute people on discord

## Commands

set Token    /setmutetoken MTIzNDU2Nzg5... (Se guarda en config/discordmute.properties)

Mute channel    /muteall channel 123456789012345678

Mute discord user    /muteall player Yuki

Unmute channel    /unmuteall channel 123456789012345678

Unmute discord user    /unmuteall player Yuki
